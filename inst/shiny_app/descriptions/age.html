Summary:
The height of the bars indicates the number of customers in the given age range.

 Tips:
To remove "(Missing)" group, activate filter by selecting all values. For example, check boxes for "Female" and "Male" on gender filter to remove entries with missing values for gender attribute.

By default, all subplots will have the same scale. The y-axis scales can be changed on the "Options" tab.

 Caution:
The number of customers in a given age group depends on how age is defined in the database. If age is based on precise data of birth and issue date, then customers may be counted multiple times if they purchasd both before and after their birthday.
